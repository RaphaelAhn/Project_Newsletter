# 🎙️ 톤 가이드 - 라디오 DJ처럼 친근하게

이 문서는 뉴스레터의 톤과 스타일을 정의합니다.

## 📻 기본 원칙

### 1. 라디오 DJ 톤
- 마치 라디오를 통해 누군가와 대화하는 느낌
- 자연스럽고 편한 느낌
- 어떤 주제든 이야기할 수 있는 친근함

### 2. 실용적이면서 친절
- 쓸데없는 말은 줄이기 (실용적)
- 하지만 따뜻한 마음으로 배려하기 (친절)
- 독자를 존중하는 톤

### 3. 쉬운 설명
- 복잡한 용어는 풀어서 설명
- 초등학생도 이해할 수 있는 수준
- "예를 들면" 같은 구체적인 사례 활용

### 4. 정확한 사실
- 팩트 체크는 절대 생략 불가
- 확실하지 않은 것은 "~라고 알려져 있어요" 표현
- 출처 명시

```

```

## ✅ 이렇게 해보세요

### 예시 1: 비트코인 뉴스

```
❌ 딱딱한 버전:
「비트코인 가격이 상승했다. 기관 투자자의 수요 증가가 주요 원인이다.」

✅ DJ 톤:
「여러분, 비트코인 얘기 나눠볼까요? 요즘 기관 투자자들이 
적극적으로 사들이고 있거든요. 쉽게 말해, 큰 손들이 
관심을 가지니까 가격도 따라가는 거죠.」
```

### 예시 2: 게임 뉴스

```
❌ 너무 어려운 버전:
「AAA 타이틀의 프로세듀럴 제너레이션 시스템이 플레이어의 
게임 플레이 루프를 최적화한다.」

✅ DJ 톤:
「신작 게임이 정말 신기한 기능을 넣었어요. 
마치 게임이 여러분의 플레이 스타일을 알아차리고 
그에 맞춰서 준비해주는 거라고 생각하면 돼요.」
```

### 예시 3: IT 뉴스

```
❌ 너무 딱딱한 버전:
「OpenAI가 새로운 멀티모달 AI 모델을 발표했다.」

✅ DJ 톤:
「OpenAI에서 정말 대단한 AI를 공개했는데요, 
이 AI는 글만 읽는 게 아니라 그림도 보고, 
영상도 이해할 수 있대요. 마치 영리한 친구가 
모든 걸 다 할 수 있는 셈이죠.」
```

### 예시 4: 축구 뉴스

```
❌ 지루한 버전:
「맨유가 새로운 공격수를 영입했다.」

✅ DJ 톤:
「맨유의 시즌이 달라질 수도 있겠다는 생각이 드는데요, 
정말 실력 있는 공격수를 데려왔거든요. 
이 선수가 적응만 잘하면 팬들이 정말 좋아할 것 같아요.」
```

## 📌 자주 사용하는 표현

**도입**
- "여러분, ~얘기 나눠볼까요?"
- "요즘 ~에서 정말 흥미로운 일이 일어났어요."
- "이번 주에 꼭 알아야 할 뉴스가 있어요."

**설명**
- "쉽게 말해~"
- "다시 말하면~"
- "예를 들면~"
- "마치 ~라고 생각하면 돼요"

**마무리**
- "앞으로 어떻게 될지 지켜볼 필요가 있을 것 같아요."
- "여러분도 어떻게 생각하세요?"
- "이런 식으로 흘러가는 것 같습니다."
