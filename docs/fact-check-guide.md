# ✅ 팩트 체크 가이드

뉴스레터의 정확성을 보장하기 위한 팩트 체크 방법입니다.

## 🎯 기본 원칙

1. **확인 가능한 출처** - 신뢰할 수 있는 출처에서 확인
2. **최소 2개 이상의 출처** - 한 곳에만 의존하지 않기
3. **최신 정보 확인** - 뉴스가 구식이 아닌지 확인
4. **공식 발표 우선** - 회사/조직의 공식 발표 확인
5. **맥락 이해** - 전체 상황을 이해한 후 전달

---

## 📚 신뢰할 수 있는 출처 (분야별)

### 게임
- ✅ 게임 개발사 공식 블로그/트위터
- ✅ Polygon, IGN, GameSpot 등 게임 매체
- ✅ Steam, PlayStation, Xbox 공식 페이지
- ❌ 개인 블로거의 추측과 예상

### IT
- ✅ TechCrunch, The Verge 등 IT 매체
- ✅ 기업의 공식 보도자료
- ✅ IEEE, ACM 같은 학술 기관
- ❌ 미확인된 루머나 추측

### 코인
- ✅ CoinMarketCap, CoinGecko 같은 데이터 사이트
- ✅ 거래소의 공식 발표 (바이낸스, 업비트 등)
- ✅ 암호화폐 관련 신뢰 매체
- ❌ 개인 트레이더의 의견이나 예측

### 축구
- ✅ ESPN, BBC Sport 같은 스포츠 매체
- ✅ 공식 클럽 홈페이지
- ✅ 연맹(FIFA, UEFA) 공식 발표
- ❌ 팬 커뮤니티의 루머

## 📋 팩트 체크 체크리스트

각 뉴스마다 이 체크리스트를 사용하세요:

```
뉴스 제목: _________________________________

[ ] 1. 공식 출처 확인
      출처: ___________________________
      URL: ____________________________
      확인 날짜: _______________________

[ ] 2. 두 번째 출처로 교차 확인
      출처: ___________________________
      URL: ____________________________
      같은 내용인가? ○ Yes  ○ No

[ ] 3. 뉴스 발표 날짜 확인
      발표 날짜: _______________________
      오래된 뉴스는 아닌가? ○ Yes ○ No

[ ] 4. 인용문 정확성 확인
      직접 인용한 부분이 정확한가? ○ Yes ○ No

[ ] 5. 맥락 확인
      전체 상황을 정확히 이해했는가? ○ Yes ○ No
      빠진 중요 정보가 있는가? ○ Yes ○ No

[ ] 6. 개인의견과 구분
      사실과 의견을 구분했는가? ○ Yes ○ No

팩트 체크 결과: ○ 통과 ○ 재검토 필요

재검토 필요 이유:
_____________________________________________
_____________________________________________
```

## 🔍 실제 팩트 체크 예시

### 예시 1: 게임 뉴스

```
뉴스: "신작 게임 XX가 출시 예정이다"

단계 1) 공식 출처 확인
→ 게임 개발사 공식 홈페이지에서 확인

단계 2) 교차 확인
→ 게임 매체 3곳에서 같은 내용 확인

단계 3) 날짜 확인
→ 최근 1주일 내 발표된 소식

단계 4) 인용문 확인
→ 개발자 인터뷰 원문과 비교

결론: ✅ 통과 - 안전하게 보도 가능
```

### 예시 2: 코인 뉴스

```
뉴스: "비트코인 가격이 50% 상승했다"

단계 1) 공식 출처 확인
→ CoinMarketCap에서 가격 데이터 확인

단계 2) 교차 확인
→ CoinGecko, 여러 거래소에서 같은 가격 확인

단계 3) 시기 확인
→ "언제" 상승했는지 명확히

단계 4) 원인 확인
→ 상승 원인이 명시되어 있는가?
→ 단순 추측인 아닌가?

결론: ⚠️ 부분 수정 필요
"50% 상승"은 정확한 수치로 수정 필요
```

## ⚠️ 주의사항

### 피해야 할 표현
- "~라고 하는데요" (불명확한 출처)
- "~라고 들었어요" (확인 안 된 정보)
- "분명~할 거야" (미래 예측을 사실처럼)
- "대부분 그렇게 생각해요" (근거 없는 주장)

### 안전한 표현
- "공식 발표에 따르면~"
- "최신 데이터에 의하면~"
- "이렇게 될 가능성이 있어요"
- "전문가들은 이렇게 보고 있어요"
